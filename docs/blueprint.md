# **App Name**: NexusTrack

## Core Features:

- Workspace Navigator: View and switch between all workspaces associated with the user, clearly indicating the currently active workspace for context.
- Workspace Dashboard: Displays key metrics and aggregated data for the active workspace, including project count, total tasks, tasks by status, and upcoming/overdue items.
- Project Lifecycle Management: Allows users to create, view, edit, and delete projects within the context of the active workspace.
- Task Canvas & Views: Facilitates creating, editing, deleting, and updating task statuses, presented in both a list view and a Kanban board.
- Contextual Search & Filter: Enables searching tasks by title and filtering by status, assignee, priority, and due date within the selected project.
- Workspace Members (Mock): Show a list of mock members within the active workspace and allow adding additional mock members for prototype demonstration.
- AI Task Detail Assistant: An AI tool to suggest and refine task descriptions, priority, or tags based on the task title and available context.

## Style Guidelines:

- Color scheme: Light theme, emphasizing clarity and focus for a productivity application.
- Primary color: A sophisticated, calming blue with a violet undertone (#452ED2), conveying professionalism and stability.
- Background color: A very subtle, cool-toned grey (#F4F3F7), derived from the primary hue, providing a clean and expansive canvas.
- Accent color: A brighter, active blue (#66A9F0), drawn from an analogous hue, for highlights, interactive elements, and calls-to-action.
- Headlines font: 'Space Grotesk' (sans-serif), for a modern, crisp, and slightly technical feel, enhancing legibility in titles.
- Body text font: 'Inter' (sans-serif), chosen for its high legibility and neutral, functional design, optimal for detailed descriptions and task content.
- Utilize a consistent set of modern, minimalist line-art icons (SVG preferred) to ensure visual clarity and contribute to the clean design aesthetic across the application.
- Adopt a multi-panel layout with a fixed left sidebar for workspace/project navigation and a fixed top bar for global actions, complementing a dynamic main content area for dashboard, project, and task views.
- Implement subtle and performant micro-interactions and transitions for core actions like task drag-and-drop, modal displays, and navigation shifts, ensuring a smooth and responsive user experience.
